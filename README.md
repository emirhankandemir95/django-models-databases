# Django Models & Databases

Practice project exploring Django ORM, models, relationships, and database queries.

## Concepts Covered
- Django model field types (CharField, IntegerField, BooleanField, DateTimeField, DecimalField)
- QuerySet filtering with filter(), exclude(), order_by(), Q objects
- Validators (MinValueValidator, MaxValueValidator)
- __str__ method implementation
- makemigrations and migrate workflow
